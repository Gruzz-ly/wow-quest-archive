    Objectives

    Search the structures in Stromgarde for Faerin and information on the Red Dawn. * Army tent searched * Inn basement searched * Tavern searched * Home searched * Mage tower searched * Faerin freed

    Description

    I saw Faerin being taken away, but couldn't follow without them seeing me. She's probably being held in another building.

    Go find her. Hopefully, she's still alive...

    And see if you can find any more information on the Red Dawn. There's no way we should have been overwhelmed so quickly. Colonel Nials and Lady Trollbane must have installed their cronies everywhere.

    The more information we have, the more we can use against them when we fight back.
