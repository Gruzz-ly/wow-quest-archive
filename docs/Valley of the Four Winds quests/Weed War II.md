{{dialogue|

This applies Weed War, a three-minute tracking buff, and starts the quest event.
{{dialogue|class=plainlist|
*{{text|yell|Gai Lan|Look, there's a weed now! Go pull it!}}
*{{text|bossemote|Click on weeds to pull them!}}

}}

Some time later, little weeds start popping up:
{{dialogue|class=plainlist|
*{{text|say|Gai Lan|Just stomp on those little weeds. No need to waste your time picking them.}}
*{{text|bossemote|Walk over small weeds!}}
}}

Later:
{{dialogue|class=plainlist|
*{{text|say|Gai Lan|Here, I've got some spare Weed-B-Gone. This should help.}}
*{{text|bossemote|Pick up the Weed-B-Gone!}}
}}
Run over, pick it up, then use the new bonus action button to bring up the targeting reticle. Aim for as many weeds as possible.

Later:
{{dialogue|
{{text|yell|Gai Lan|Look out!}}
}}
