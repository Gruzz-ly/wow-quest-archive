==Objectives==
Use a First Aid Kit on Austin Huxworth, Bjorn Stouthands, and Kee-la.
* First Aid Kit used on [[Bjorn Stouthands]]
* First Aid Kit used on [[Kee-La]]
* First Aid Kit used on [[Austin Huxworth]]
* [[First Aid Kit (2)]] (provided)

==Description==
We have to patch everyone up before we can begin the search for the missing expedition.

Take this first aid kit and tend to the wounded.

==Rewards==
