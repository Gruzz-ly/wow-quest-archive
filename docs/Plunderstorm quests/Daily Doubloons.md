The '''Daily Doubloons''' is a daily world quest that provides 800 [[Plunder]] for completing the first [[Captain's Orders]] of the day which also provides 250 Plunder for a total of 1050 Plunder for the daily.

== Objectives ==
*Captain's Orders completed

You will receive:
