== Description ==

== Rewards ==
You will receive:
* 25 [[Legionfall War Supplies]]
* Varies
* +150 reputation with [[Armies of Legionfall]]

== Completion ==
: {{text|say|Illidan Stormrage|You have hindered the [[Burning Legion|Legion's]] war production, if only briefly. Every victory counts on this dark shore.}}

== Notes ==
Triggers at [[Felfire Pass]] on the [[Broken Shore]]. [[Felforge Laborer]]s can be killed and [[Fel Crucible]]s clicked on for 2-4 pieces of felslag. There are also single pieces that can be picked up off the ground.
