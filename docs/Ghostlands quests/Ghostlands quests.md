# Ghostlands quests

This is a list of all [[Ghostlands]] quests in table format.

For a questing guide detailing the best order to obtain and complete quests in [[Ghostlands]], see [[Ghostlands questing guide]].

[[Delivery to Tranquillien]] introduces the young [[Blood Elf]] players to this region.

==Quest table==
{{See also|Ghostlands questing guide}}
Quests which require players to leave [[Ghostlands]] are denoted by a {{Outzone}}. Mouse over the icon to see where the quests take place.

Quests which are part of a chain are denoted by {{Chain}}. Click the downward arrow to the right of the icon to view the chain.
