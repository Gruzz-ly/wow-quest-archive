== Objectives ==
Deliver the bad news to [[Elder Atuik]] at [[Kamagua]].

== Description ==
Return to Elder Atuik at Kamagua, far to the west. He and the rest of my people must be warned of this impending doom that approaches from the sea.

<[[Orfus of Kamagua|Orfus]] shivers.>

Use the [[Ancient Lift]] to reach him.  Please hurry!
