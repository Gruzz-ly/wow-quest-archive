'''Delves: Worldwide Research''' is one of several meta quests that can pop up each week in [[Dornogal]].

==Objectives==
Complete 3 Delves in Khaz Algar.
*Complete any [[delve]] in [[Khaz Algar]]

==Description==
It's been all quiet recently. I've nae heard of any issues.

This seems like a good opportunity for exploration, eh? I'm at yer disposal! We can go wherever it is ye want tae go!
