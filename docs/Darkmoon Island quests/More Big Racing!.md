==Objectives==
Run through the Start Banner, hit the Purple Post, then the Green Post, then the Red Post, then the return to the Start Banner.
*Enter the Start Banner with a Darkmoon Game Token
*Reach the Purple Post
*Reach the Green Post
*Reach the Red Post
*Return to the Start Banner
*Race Complete

==Description==
You like this big race, do you? Me too!

It has four legs: get to the purple post, then green, then red, then back here.

Your time matters, so do it fast!

==Completion==
Good job, and nice time!
