== Objectives ==
* Fly a [[Lightforged Warframe|Warframe]] into [[Antorus]]
* [[Spire of Woe]] destroyed (3)
* Demons slain (15)
* [[Sentinax]] destroyed

== Description ==
: {{text|say|[[Grand Artificer Romuul]]|Forces inside Antorus must see they are not safe from our attacks. Take a warframe and hit targets of opportunity behind their lines!}}

== Completion ==

== Gains ==
* +75 reputation with [[Army of the Light]]

== Notes ==
Note that kills will not start counting until the first criteria is complete, by reaching [[Collector's Hovel]]. Think of them as shooting practice, if you like.
