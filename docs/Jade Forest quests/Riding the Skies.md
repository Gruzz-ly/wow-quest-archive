==Description==
I can no longer call myself your teacher, <name>.

You have proven yourself worthy of your cloud serpent.

Today, you shall join our ranks. Are you ready?

==Rewards==
{{itembox|You will receive one of:|Reins of the Jade Cloud Serpent|Reins of the Azure Cloud Serpent|Reins of the Golden Cloud Serpent}}
You will also receive:
*{{cost|9|92|50}}
*118000 [[XP]]

==Completion==
Today, we become brethren.  Congratulations on joining the Order of the Cloud Serpent.  Let us ride together!

==Notes==
There are three versions of this quest, one for each possible color of cloud serpent awarded. The character will always receive the serpent corresponding to their choice of egg in [[Choosing the One]], so only that version of the quest will be offered:
* [[Reins of the Azure Cloud Serpent]] if [[Blue Serpent Egg]] was chosen
* [[Reins of the Golden Cloud Serpent]] if [[Yellow Serpent Egg]] was chosen
* [[Reins of the Jade Cloud Serpent]] if [[Green Serpent Egg]] was chosen
