==Rewards==

*+75 [[reputation]] with [[Valarjar]]

==Notes==
Drink mead, throw plates, start some fisticuffs, make some toasts, or defend yourself from fisticuffs, it's all good.

If you do decide to drink, and get drunk enough, you'll be able to see [[Stormy Wolpertinger]]s that can be kicked for progress.
