== Objectives ==
Rescue 5 Horde Prisoners.
*{{NPC||Horde Prisoner (Abyssal Depths)||Horde Prisoner}} rescued x5

== Description ==
The [[Twilight's Hammer|Twilight cultist]] dropped this key. Use it to free the [[Horde]] prisoners in the cages inside of {{NPC||L'ghorek}}.

Hurry before the prisoners become sacrifices to whatever lurks below!

== Rewards ==
{{itembox|You will be able to choose one of these rewards:|Darkspear Ritual Blade|Goblin Shanker|Grimhorn Pummeler}}

== Completion ==
A job well done. Now you don't need to worry about what would have happened to them if you hadn't been around.

The last one was even appreciative enough to give you a reward they'd kept on their person.

== Notes ==
While on the quests [[Runestones of Binding]] and [[Ascend No More!]] inside of [[L'ghorek]], a Twilight's Hammer cultist will drop the Twilight Cage Key, which starts this quest. Free prisoners in cages in the Twilight camp in the southwestern portion of L'ghorek's main chamber.
