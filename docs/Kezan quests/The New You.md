== Objectives ==
Buy some Shiny Bling from {{NPC||Gappy Silvertooth}}, a Hip New Outfit from {{NPC||Szabo}}, and some Cool Shades from {{NPC||Missa Spekkies}} on [[Swindle Street]].

== Description ==
A new outfit? You don't say! There's quite a buzz surrounding your party. Everyone who's anyone on [[Kezan|the island]] is going to be there.

Well, happy shopping. Swindle Street has some of the best bargains anywhere!
