==Objectives==
Install the [[Rustbolt Kegerator]] in [[Rustbolt]]'s bar, The [[Tipsy Tinkerer]].

==Description==
The massive kegerator that you built should be able to house all of the [[Rustbolt Resistance]]'s beverages.

Find a spot to set it up in Rustbolt, and let the town know it's ready for use!

==Rewards==
You will receive:
* +500 reputation with [[Rustbolt Resistance]]

==Completion==
This looks like a great place to install the kegerator.
