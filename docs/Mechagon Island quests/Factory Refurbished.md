==Description==
I need you to get us a Recycled Crawler Manifold.

You should be able to get one up at [[Junkwatt Depot]] if you recycle parts found there.

==Rewards==

==Progress==

==Completion==

== Notes ==
If you haven't already done so, you will complete [[Recyclable Parts]] with this quest from [[Recyclerizer DX-82]] and unlock the dailies [[More Recycling]] (gives rep) and [[Even More Recycling]] (no rep).

You will need in total:

* [[Hardened Spring]] x30
* [[Tempered Plating]] x10
* [[Machined Gear Assembly]] x5

Complete Recyclable Parts/More Recycling/Even More Recycling for a [[Recycling Requisition]], which will contain the manifold.
