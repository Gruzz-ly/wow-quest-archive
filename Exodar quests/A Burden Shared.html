    {{Questbox | id = 78082 | name = A Burden Shared | category = Heritage | faction = Alliance | level = {{Level|Heritage}} | race = Draenei | start = [[Prophet Velen]] {{Co|54.93|80.73|Exodar}} | end = [[Grand Anchorite Almonen]] {{Co|64.37|87.02|Exodar}} | rewards = 10x [[Flightstones]] | experience = 6,800 | money = {{cost|23|40}} | previous = [[Pain Recedes]] | next = [[Our Path Forward]] }}

    ==Objectives== Collect 10 Memory Stones, create your own Memory Stone, and then dissolve the stones in the [[Flame of Reprieve]]. Memory Stones collected (10) Personal Memory Stone created *Memory Stones dissolved

    ==Description== The [[Tishamaat]] was once a solitary holiday. Each [[eredar]] would inscribe a stone with a painful memory we carried. We would reflect on what that memory could teach us then commit to moving forward with that wisdom.

    At last, the stones are dissolved in the Flame of Reprieve, symbolically removing the weight from our souls.

    Today, the Tishamaat will be celebrated not in solitude, but as a community.

    Gather the stones of those who would trust you with them then take them to the flame.

    ==Rewards== You will receive: {{cost|23|40}} 6,800 [[XP]] *10x [[Flightstones]] (max level)

    ==Progress== For many, this is the first Tishamaat they have participated in. For others, it is a distant memory.

    ==Completion== The [[draenei]] have suffered, scattered and solitary, for too long.

    Today, we allow ourselves to rely on each other.
