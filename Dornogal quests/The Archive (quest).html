    The Archive (quest)

    {{Questbox | id = 78468 | name = The Archive | category = Isle of Dorn | level = {{level|Isle of Dorn}} | start = [[Speaker Brinthe]] {{co|29.4|59.9|Dornogal}} | image = The Archives.jpg | experience = 5,200 | money = {{Cost|14|4}} | shareable = Yes | previous = [[Hypocenter]] | next = [[State of the Union]] }}

    ==Objectives== Use the Prismatic Codex to view the Memory Gem. Then, return the Memory Gem to the Archival Repository in The [[Archive]]. Prismatic Codex used to view [[Earthen Memory Gem|memory gem]] Memory gem deposited in [[Archival Repository]]

    ==Description== The memory gem you recovered is what its name suggests. It contains an [[earthen]]'s memories, directives, and a hard coding of our Edicts.

    This building, the Archive, is the nexus of our civilization. Here, memories are cycled, catalogued, and stored at the end of an earthen's scheduled rotation.

    With the codex I have invented, we can access one or two memories from a memory gem. Take it, and read the memory gem you found.

    ==Rewards== You will receive: {{Cost|14|4}} 5,200 [[XP]]

    ==Progress== The operating instructions are simple. Insert the gem into the device, and a memory will play.

    ==Completion== It is... comforting that Odon will be preserved in the Archive.
